<============== REQUIREMENTS ==============>
Python - 3.6.5
Django - 2.1a1

<============== INSTALLATION ==============>

To install python log onto : https://www.python.org/
To install Django:
------ In Windows: ------
1> Open PowerShell as administrator.
2> execute command : easy_install django
------ In Linux/Unix: ------
*** Perform above steps using 'sudo' command in Terminal ***
3> To verify installation perform : django-admin --version

<============== EXECUTION ==============>
1> Change directory to the appropriate project path.
2> To initiate server process execute command : python manage.py runserver
3> Open the localhost url : http://127.0.0.1:8000/ in browser.
